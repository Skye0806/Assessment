Instructions to Edit the assessments:

Folder OrgCode contains files to provide assessments that are anonymous and dont need login info. 

Folder loginAssessment contains files for assessments after login.

To update Assessment name:

Edit index.js file in assets folder which has a commented line. 

Example below:

//Enter Assessment Name
var assessment = "AssesmentTest";

to change Assessment Test to STEMAssessment please see below

//Enter Assessment Name
var assessment = "STEMAssessment";


Other changes include updating front end instructions and options. 

Please see index.html file to be able to change that info. 


INORDER TO AVOID QUESTIONS BEING UPDATED EVERYTIME AN ASSESSMENT IS SUBMITTED PLEASE COMMENT OUT THE JS CODE INDICATED IN INDEX.JS FILE UNDER ASSESTS FOLDER
